监听移动端软键盘弹起和收起

Mar 2, 2017 | Hits

因为ios和安卓显示软键盘是不一样的
1.ios是软键盘是以一个浮层的形式显示在页面上,所以它会影响网页上的布局,如果底部有个类似fixed的元素会受到影响…在对待软键盘上,点击input弹出软键盘,点击完成或者点击别处就会自动失焦,而安卓不会,仍旧停留在input上.
2.安卓系统一部分手机跟ios显示的一样,但大部分手机是键盘向上推,把窗口缩小
